## Описание

TheСostOfTripsTestTask - Angular приложение написанное в качестве тестового задания.

## Использованные технологии

+ `Angular` - https://github.com/angular/angular
+ Язык - `TypeScript` - https://github.com/Microsoft/TypeScript


## Установка

```bash
$ npm install
```

## Запуск

```bash
# development
$ npm run start

#build
$ npm run build
```
